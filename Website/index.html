Getting Started

  1. Click Add to VCC below to add this listing
  2. In VCC, add Dependency Checker Setup to your project
  3. In Unity, open Terrorized/Installers/Dependency Checker Setup from the top menu
  4. Enter your license key and click Verify License
  5. Set your creator name, splash, socials and dependencies, then click Download and Install
