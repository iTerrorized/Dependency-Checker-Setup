# Dependency Checker Setup

A tool for VRChat avatar creators to ship with their avatars to ensure customers install the correct dependencies for the project. The Dependency Checker verifies that a buyer has the correct Unity version, the correct build platform, and all required VPM packages. It also helps them install anything that is missing.

## Requirements

- Unity 2022.3
- dev.terrorized.drm (added automatically when you install through the listing below)

## Installing

Add the listing in the VRChat Creator Companion or ALCOM:

https://iterrorized.github.io/Dependency-Checker-Setup/

Then add Dependency Checker Setup to your project and open it from Terrorized/Installers/Dependency Checker Setup at the top of the Unity editor.
